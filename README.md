# Modeling a Kanban Production System

MATLAB/Simulink model of a Kanban production system with performance analysis and optimization.

## Overview

This example shows a production system that uses kanbans to manage production activities. Analysis of simulation results highlights problems in the system and suggests ways to improve its performance.

## Files

- `KanbanSystemExample.m` - MATLAB script for running the Kanban system simulation
- - `seKanbanSystem.slxc` - Simulink model cache file
  - - `Modeling a Kanban Production System.docx` - Documentation in Word format
    - - `Modelling a Kanban Production System.pdf` - Documentation in PDF format
